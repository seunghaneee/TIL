## CLI

##### GUI vs CLI

> why CLI

- GUI는 CLI에 비해 사용하기 쉽지만 단계가 많고 컴퓨터의 성능을 더 많이 소모
- 수 많은 서버/개발 시스템이 CLI를 통한 조작 환경을 제공

> 기본적인 명령어

- touch
  - 파일을 생성하는 명령어
- Mkdir
  - 새 폴더를 생성하는 명령어
- cd
  - 현재 작업 중인 디렉토리를 변경하는 명령어
- start, open
  - 폴더/파일을 여는 명령어(windows 에서는 start, Mac에서 open을 사용)
- rm
  - 파일을 삭제하는 명령어
  - -r 이용하면 삭제
